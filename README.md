**JobVista** is a comprehensive job seeking and recruitment platform built with the MERN stack.
It facilitates a seamless connection between job seekers and employers, offering advanced features like real-time application tracking, resume management, and admin dashboards.

---

## 🌟 Key Features

### 👨‍🎓 For Job Seekers (Students)
* **Advanced Search:** Filter jobs by keywords, location, and salary.
* **One-Click Apply:** Easy application process with profile auto-fill.
* **Resume Management:** Upload and update resumes (PDF support via Cloudinary).
* **Application Tracking:** Real-time status updates (Applied, Accepted, Rejected).
* **Profile Customization:** Edit skills, bio, and contact information.

### 👨‍💼 For Recruiters (Admins)
* **Job Management:** Create, edit, and delete job postings.
* **Applicant Dashboard:** View a table of all applicants for specific jobs.
* **Resume Viewer:** Direct links to applicant resumes.
* **Status Control:** Update application status to shortlist or reject candidates.
* **Company Profiles:** Register and manage company details.

## 🛠️ Tech Stack

| Layer | Technology |

| **Backend** | Node.js, Express.js |
| **Database** | MongoDB (Mongoose ODM) |
| **Authentication** | JWT (JSON Web Tokens), Bcrypt.js |
| **File Storage** | Cloudinary (for Images & PDFs) |
| **Tools** | Postman, VS Code, Git |
